# LogAtlas · Intelligent Logistics Sorting Robot

> Promotional website for the LogAtlas intelligent logistics sorting robot (B39VS Systems Project).

## How to Open

1. Unzip this folder to any location
2. **Double-click `index.html`** to open in your default browser
3. No server, no internet, no installation required — fully offline

## Folder Structure

```
logatlas-offline/
├── index.html        ← Main page (double-click to open)
├── styles.css        ← Custom styles
├── script.js         ← Interactions & animations
├── fonts.css         ← Font definitions (local)
├── README.md         ← This file
├── assets/           ← Product images
│   ├── logo.png
│   ├── robot-hero.png
│   └── arm-cad.png
├── fonts/            ← Font files (Inter, Space Grotesk, JetBrains Mono)
│   └── font_*.ttf
└── lib/              ← JavaScript libraries
    ├── tailwind.js   ← TailwindCSS (JIT compiler)
    └── lucide.js     ← Lucide icon library
```

## Browser Compatibility

- Chrome 90+ (recommended)
- Edge 90+
- Firefox 90+
- Safari 15+

## Tech Stack

- **TailwindCSS** — Utility-first CSS (JIT, runs in-browser)
- **Lucide** — Icon library
- **Inter / Space Grotesk / JetBrains Mono** — Typography

Project Showcase

Three industrial challenges,
three breakthroughs.

We don't shy away from the hard parts. For each known failure mode, we rebuild the math, the firmware, and the interface — until the problem is gone.

01 / Navigation

Autonomous Navigation

Challenge

Low-level sensor navigation in grid topologies suffers a 12–15% routing error rate without global orchestration.— IEEE Trans. Automation Sci. & Eng., 2021

Our Solution

Cloud-Edge decoupling. An Android App maintains DAG node memory via BLE; an Arduino edge node runs an 8-ch IR array + dual-wheel PID for local attitude correction.

02 / Manipulation

Container Grabbing

Challenge

Single-loop PID can produce overshoot up to 20% in micro-operations and settling times over 8 seconds.— J. of Dynamic Systems, 2019

Our Solution

Decoupled architecture on DS-SY15A servos (20 kgf·cm, 115200 baud). Outer loop tracks trajectory, inner loop absorbs high-frequency vibration — the arm gains critical damping.

03 / Safety

Obstacle Avoidance

Challenge

Dead-reckoning obstacle bypass has a trajectory recovery failure rate of up to 25%.— Robotics and Autonomous Systems, 2020

Our Solution

The bypass maneuver is abstracted as a time-based geometric mapping. HC-SR04 triggers at <15 cm; HAL_GetTick() differential (ms-precision) drives reverse-integral blind merging.
